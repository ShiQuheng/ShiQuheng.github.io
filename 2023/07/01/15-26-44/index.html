MySQL安装教程

发表于 2023-07-01 更新于 2023-07-03 分类于 MySQL

摘要
本文记录了MySQL的安装教程。

MySQL下载与安装

一、下载

  1. 下载地址：*https://downloads.mysql.com/archives/community/*
  2. 当前最新是8.0版本，我选择上一个最新的mysql-5.7.24-winx64.zip

二、安装

  • zip格式是自己解压，解压缩之后其实MySQL就可以使用了：
  • bin目录下保存了MySQL常用的命令工具以及管理工具。
  • data目录是MySQL默认用来保存数据文件以及日志文件的地方（我的因刚安装还没有data文件夹, 后面运行mysqld --initialize-insecure --user=mysql时会自动创建data文件夹
    ）。
  • docs目录下是MySQL的帮助文档。
  • include目录和lib目录是MySQL所依赖的头文件以及库文件。
  • share目录下保存目录文件以及日志文件。

但是要进行环境变量配置zip格式是自己解压。

  1. 我的电脑->属性->高级系统设置->环境变量

  2. 选择Path,添加: 你的mysql bin文件夹的路径;
    E:\Java\MySQL\mysql-5.7.24-winx64\mysql-5.7.24\bin

  3. 配置完环境变量之后，在E:\Java\MySQL\mysql-5.7.24-winx64\mysql-5.7.24目录下新增加一个配置文件mysql.ini

  4. 编辑自己的mysql配置文件, 在mysql.ini文件里添加如下内容：

    注意修改其中的安装目录以及数据存放目录为自己的目录。

  5. 使用管理员模式打开cmd,，不需要进入安装目录（因为之前配置过环境变量），输入下面命令:
    mysqld --initialize-insecure --user=mysql
    回车：

    用于初始化MySQL数据目录并指定MySQL用户。这会在安装目录下新建一个data文件夹。

    • 出现下面的错误是因为自己新建了data文件夹，导致初始化失败。

      解决方法就是把data文件夹删了再执行上面的命令即可。

  6. 输入下面命令：mysqld install
    回车，会提示安装成功:

  7. 启动服务，输入如下命令:net start mysql，回车，启动成功后如下图:

  8. 服务启动成功之后，需要登录的时候输入命令:mysql -u root -p（第一次登录没有密码，直接按回车过）

  9. 修改密码（先退出刚刚登录的MySQL，输入\q即可退出(注意直接关闭cmd是不会退出的，要输入退出指令\q或者exit)，然后再启动mysql，输入net start mysql），执行如下命令：mysqladmin -u root -p password，回车，enter password也回车，密码一般设置为root，方便记忆,这里我设置为123456：

  10. 改完密码后重启MySQL，再试试登录MySQL:
    命令行输入：mysql -u root -p123456注意-p后面不要加空格！！！

关闭mysql开机自启动

  1. 在运行中输入:services.msc(windows快捷键win+R)

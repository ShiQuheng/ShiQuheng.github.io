LeetCode-替换空格

发表于 2023-03-08 更新于 2023-03-10 分类于 LeetCode

摘要
把字符串s中的每个空格替换成”%20”。

题目描述

实现一个函数，把字符串s中的每个空格替换成"%20"。
示例 1：

输入：s = “We are happy.”
输出：”We%20are%20happy.”

限制：

0 <= s 的长度 <= 10000

解法

方法一：字符串内置方法

使用 replace() 方法。
时间复杂度$O(n)$，空间复杂度$O(n)$。其中$n$为字符串长度。

方法二：遍历替换

我们直接遍历字符串，遇到空格就替换成%20即可。

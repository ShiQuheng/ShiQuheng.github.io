Java命名规则

发表于 2023-03-22 更新于 2023-03-23 分类于 Java

摘要
本文记录了Java程序命名规则。

所有变量、方法、类名：需符合 见名知意 原则。

类成员变量

类成员变量：首字母小写和驼峰原则:除了第一个单词以外，后面的单词首字母大写。
例如：

monthSalary、lastName

局部变量

局部变量需满足：首字母小写和驼峰原则。
例如：

lambdaValue、numEpisode

常量

规则：大写字母和下划线。
例如：

PI、MAX_VALUE

类名

规则：首字母大写和驼峰原则。
例如：

Man、GoodMan

方法名

规则：首字母小写和驼峰原则。
例如：

run()、saveCheckpoint()

包名

规则：一般使用公司域名倒置。
例如：

www.baidu.com com. baidu. XXX

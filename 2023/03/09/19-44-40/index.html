LeetCode_从尾到头打印链表

发表于 2023-03-09 更新于 2023-03-10 分类于 LeetCode

摘要
从尾到头打印链表

题目描述

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。
示例 1：

输入：head = [1,3,2]
输出：[2,3,1]

限制：

0 <= 链表长度 <= 10000

解法

方法一：顺序遍历 + 反转（栈）

我们可以顺序遍历链表，将每个节点的值存入数组中，然后将数组反转。可以用栈来实现。
时间复杂度$O(n)$，空间复杂度$O(n)$。其中$n$是数组链表的长度。

方法二：递归

我们可以使用递归的方式，先递归得到head之后的节点反过来的值列表，然后将head的值加到列表的末尾。
时间复杂度$O(n)$，空间复杂度$O(n)$。其中$n$是数组链表的长度。

扩展Python切片

在 Python 中，a[::-1]是一种切片语法，用于反转列表a中的元素顺序。
切片语法的一般形式为a[start:stop:step]，其中start是切片的起始索引, stop是切片的结束索引（不包括），而step是切片的步长。当省略start和stop时，默认值分别为列表的起始和结束索引。当省略step时，默认值为1。
因此，当你使用a[::-1]时，相当于从列表的末尾开始，以步长为-1（即反向）遍历整个列表。这样就可以得到一个反转后的新列表。
